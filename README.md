# Databricks Sales Dashboard Demo

Interactive business intelligence dashboard built using Databricks SQL and the Databricks sample Bakehouse dataset.

## Dashboard Preview

![Dashboard Overview](images/dashboard-overview.png)

## Features

- Product sales analysis
- Payment method breakdown
- Sales trends over time
- Interactive dashboard filters
- Built using Databricks SQL

## Technologies

- Databricks SQL
- SQL
- Data Visualization
- Business Intelligence
- GitHub

## Dataset

This project uses the built-in Databricks sample dataset:

`samples.bakehouse.sales_transactions`

## Skills Demonstrated

- SQL aggregations (`SUM`, `COUNT`)
- `GROUP BY`
- `ORDER BY`
- Dashboard design
- Interactive filtering
- Data visualization
