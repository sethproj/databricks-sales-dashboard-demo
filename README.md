# Databricks Sales Dashboard Demo

Interactive business intelligence dashboard built using Databricks SQL and the Databricks sample Bakehouse dataset.

## Features

- Product sales analysis
- Payment method breakdown
- Sales trends over time
- Interactive dashboard filters
- Built using Databricks SQL
